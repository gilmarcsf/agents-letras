Classify the product before choosing style.

- **SaaS / B2B software**
- **Developer tool**
- **Analytics / BI / operations**
- **Finance / banking / insurance**
- **Healthcare / wellness**
- **Commerce / marketplace**
- **Booking / appointment / local service**
- **Media / music / entertainment**
- **Creative tool / portfolio / agency**
- **Education / knowledge / productivity**
- **AI / automation / agent platform**
- **Community / social / creator economy**
- **Travel / hospitality / events**
- **Real estate / automotive / high-consideration purchase**
- **Public service / nonprofit / civic**
- **Logistics / IoT / field operations**

Each category changes what users expect from hierarchy, tone, density, motion, and trust cues.

Useful category splits:

- Finance, insurance, healthcare, legal, and public service require visible trust, plain language, strong contrast, and conservative motion.
- Analytics, logistics, IoT, admin, and operations require scanability, stable navigation, clear status, filters, and resilient empty/loading/error states.
- SaaS, developer tools, AI tools, and productivity products usually need a product demo or working surface before brand atmosphere.
- Commerce, marketplace, booking, real estate, automotive, hospitality, and events need strong visual proof plus a low-friction conversion path.
- Media, music, gaming, fashion, creator, and portfolio products can use stronger identity, richer motion, and more expressive typography.
